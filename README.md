# 🚀 Jornada PHP - Ensino Médio

Bem-vindo ao meu repositório de estudos! Este espaço é dedicado à documentação da minha jornada de aprendizado em **PHP** durante o ensino médio(especificamnte na 12a Classe). Aqui, centralizo todas as tarefas, exercícios e projetos práticos desenvolvidos em aula.

## 📌 Sobre o Projeto
Este repositório serve como um **arquivo pessoal e portfólio**. O objetivo é registrar minha evolução técnica, desde scripts básicos até sistemas com integração de banco de dados e interfaces modernas.

### 🛠️ O que você encontrará aqui:
- **Hub Central**: Um portal unificado para acessar todos os subprojetos.

## ⚠️ Aviso de Uso
> [!IMPORTANT]
> Este é um repositório de caráter **estritamente pessoal e educativo**.

> - **Visualização**: Sinta-se à vontade para explorar o código e ver minha evolução.

> - **Contribuições**: Não estou aceitando Pull Requests ou alterações externas neste projeto, pois ele deve refletir apenas o meu trabalho individual na escola.

> - **Uso**: Proibida a replicação para fins comerciais sem autorização.

## 💻 Tecnologias Utilizadas
- **Backend**: PHP 8.x

- **Banco de Dados**: PostgreSQL

- **Frontend**: HTML5, Vanilla CSS, Tailwind CSS (em componentes específicos)

- **Segurança**: Gerenciamento de variáveis de ambiente via `.env`
